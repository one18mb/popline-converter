# PopLine Converter

PopLine ↔ JSON / YAML / TOML / INI / XML 在线格式转换器。纯前端，零后端，GNU 简约风格。

## 使用

直接访问 GitHub Pages 链接，或本地运行：

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build     # 输出到 dist/
npm run preview   # 预览构建结果
```

## 测试

```bash
npm test
```

## 技术栈

TypeScript + Vite + [popline-js](https://www.npmjs.com/package/popline)
