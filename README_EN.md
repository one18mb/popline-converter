# PopLine Converter

Online format converter: PopLine ↔ JSON / YAML / TOML / INI / XML. Pure frontend, zero backend.

## Usage

Try it online: [https://one18mb.github.io/popline-converter/](https://one18mb.github.io/popline-converter/)

Run locally:

```bash
npm install
npm run dev
```

## Build

```bash
npm run build     # output to dist/
npm run preview   # preview build output
```

## Test

```bash
npm test
```

## Tech Stack

TypeScript + Vite + [popline-js](https://www.npmjs.com/package/popline)
